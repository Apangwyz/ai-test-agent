## 核心模块

- - **监管就绪（Regulation-First）**：所有组件默认启用等保三级配置模板（如TLS 1.3强制、密码套件白名单、日志保留≥180天）。
- - **双栈并行（Dual-Stack）**：生产环境主栈为信创栈（Q3闭环），公有云为灾备/灰度栈；**代码零差异，配置驱动适配**。
